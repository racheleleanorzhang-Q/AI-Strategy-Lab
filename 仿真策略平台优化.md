## 关键设计决策

| 决策 | 方案 | 理由 |
|------|------|------|
| 规则引擎是否保留？ | 保留为默认 | 快速、透明、可解释 |
| 训练数据来源？ | 先用规则引擎生成合成数据 | 冷启动方案 |
| 前端还是后端跑模型？ | 后端 FastAPI | 浏览器跑深度学习不现实 |
| 模型切换后怎么对比？ | 同一参数多模型跑，并排展示 | 体现预测差异 |
| Phase 1/2 部署？ | Next.js API Routes 单部署 | 零额外成本 |
| Phase 3+ 部署？ | FastAPI on Modal/Render | GPU 需求 |
