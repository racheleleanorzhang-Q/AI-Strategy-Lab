# 🧠 AI Strategy Simulator (P0)

一个基于"世界模型"的策略模拟台
用于在上线前推演 AI 产品策略（定价 / 免费额度 / 模型成本）

## ✨ 能力

- 模拟用户行为（留存 / 使用）
- 模拟AI成本结构
- 对比不同策略收益
- 输出收入 / 成本 / 利润 / 留存

## 🎯 适用场景

- AI产品定价策略
- Agent商业模型设计
- 增长策略推演
- 投资/产品演示
